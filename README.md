# Databricks Curriculum Analytics Project
AI‑generated prerequisite data → SQLite (cityu.db) → Spark DataFrames → (Delta pending) → SQL analytics

This repository contains the Databricks component of a larger, ongoing curriculum‑analysis system.
It loads scraped + AI‑inferred course data from a SQLite database (`cityu.db`), converts it into
Spark DataFrames, and performs distributed SQL analytics to understand curriculum structure.

Databricks Ingestion Code
------------------------------------------------------------

import sqlite3
import pandas as pd
import os

volume_file_path = "/Volumes/workspace/default/ds625_tp/cityu.db"

if not os.path.exists(volume_file_path):
    raise FileNotFoundError("File not found at the expected path.")

conn = sqlite3.connect(f"file:{volume_file_path}?mode=ro", uri=True)

courses_pd = pd.read_sql_query("SELECT * FROM courses;", conn)
prereqs_pd = pd.read_sql_query("SELECT * FROM prerequisites;", conn)
conn.close()

spark_courses = spark.createDataFrame(courses_pd)
spark_prereqs = spark.createDataFrame(prereqs_pd)

SQL Analytics
------------------------------------------------------------

A. Courses With vs Without Prerequisites

SELECT 
    CASE WHEN p.course_code IS NOT NULL THEN 'Has Prerequisites'
         ELSE 'No Prerequisites Mentioned' END AS Status,
    COUNT(DISTINCT c.code) AS Total_Courses
FROM spark_courses c
LEFT JOIN spark_prereqs p 
    ON c.code = p.course_code
GROUP BY Status;

------------------------------------------------------------

B. Prerequisite List Per Course

SELECT 
    c.code AS Course_Code,
    c.title AS Course_Title,
    IFNULL(CONCAT_WS(', ', COLLECT_LIST(p.prereq_code)), 'None') AS Prerequisite_List
FROM spark_courses c
LEFT JOIN spark_prereqs p 
    ON c.code = p.course_code
GROUP BY c.code, c.title;

------------------------------------------------------------

C. Credit Distribution by Department

SELECT 
    REGEXP_EXTRACT(code, '^([A-Z]+)', 1) AS Department,
    COUNT(code) AS Total_Courses,
    ROUND(AVG(credits), 2) AS Avg_Credits
FROM spark_courses
GROUP BY Department;

------------------------------------------------------------

D. Bottleneck Courses (Most Depended On)

SELECT 
    p.prereq_code AS Prerequisite_Course, 
    COUNT(p.course_code) AS Dependent_Courses
FROM spark_prereqs p
GROUP BY p.prereq_code
ORDER BY Dependent_Courses DESC
LIMIT 10;
